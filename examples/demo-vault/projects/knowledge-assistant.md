# Knowledge Assistant Demo 0

Demo 0 validates typed transcript retrieval before ASR. It indexes a sample Obsidian vault, ranks related notes, explains match reasons, and opens notes through Obsidian URI links. The target latency is under 200 ms P95 for local retrieval.
